# Local Workspace Frontend Package

The reusable class-management frontend starts as a local workspace React package instead of a copy-paste source kit or an npm package. It exposes both a **Headless Core** and ready-made **Workflow Components**, and those components use a host-supplied **UI Primitive Adapter** for ShadCN-compatible primitives so Eden can prove the package boundary now while future Consumer Websites can customize presentation without forking product behavior.
